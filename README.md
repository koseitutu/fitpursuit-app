# FitPursuit — AI Home Gym Tracker & Coach

A specialized mobile application blueprint built in **React Native (Expo)** designed to act as an automated fitness coach and dynamic tracking log tailored specifically for an all-in-one multi-gym station.

## 🚀 Key Features Include
- **5-Day Full-Body Matrix:** Hardcoded workout templates (Monday to Friday) targeting strategic muscle splits.
- **Interactive Instruction Panels:** Instant expandable guides mapping layout configurations and step-by-step physical mechanical patterns for 17 unique station attachments.
- **Smart Rest-Timer Overlay:** Built-in automatic 60-second rest interval timer triggered immediately upon logging sets.
- **AI Coach Logic Engine:** Programmed automated processing checks inspecting completion rates. If target weights or max thresholds are achieved, an integrated logic trigger prompts custom personal trainer recommendations for progressive overload.
